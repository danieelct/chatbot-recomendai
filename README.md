# 🤖 RecomendaAI - Chatbot de Entretenimento

O **RecomendaAI** é um assistente virtual inteligente desenvolvido para fornecer recomendações personalizadas de filmes, séries e músicas. O projeto utiliza inteligência artificial generativa para interagir com o usuário de forma natural e contextual.

## 🚀 Demonstração
Você pode testar a aplicação rodando ao vivo aqui: 
[Clique aqui para acessar o RecomendaAI](https://danieelct.pythonanywhere.com)

---

## 🛠️ Tecnologias Utilizadas

* **Python 3.11**: Linguagem principal.
* **Flask**: Micro-framework web para o backend.
* **Google Gemini API**: Modelo de linguagem (LLM) para as respostas da IA.
* **HTML/CSS**: Interface do usuário (Front-end).
* **PythonAnywhere**: Hospedagem da aplicação.

---

## 📋 Funcionalidades

* Respostas em tempo real baseadas em prompts otimizados.
* Interface amigável e responsiva.
* Memória de contexto (mantém o histórico da conversa durante a sessão).
* Sugestões específicas por gênero, humor ou plataforma de streaming.

## 🔧 Como Rodar o Projeto Localmente

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/danieelct/chatbot-recomendai.git

2. **Instale as dependências:**

Bash
pip install -r requirements.txt

3. **Configure sua Chave de API:**
Crie um arquivo .env na raiz do projeto e adicione sua chave do Gemini:

GEMINI_API_KEY=SUA_CHAVE_AQUI

4. **Execute o app:**

Bash
python app.py
